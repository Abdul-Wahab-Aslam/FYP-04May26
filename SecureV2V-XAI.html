Problem Statement

Modern V2V communication networks face critical vulnerabilities from adversarial attacks including Sybil, replay, GPS spoofing, and DoS flooding. Traditional cryptographic protocols are insufficient against AI-driven threat actors in real-time vehicular environments.

Objectives
  • Real-time AI security layer for V2V protocols
  • Detection latency <30ms on edge hardware
  • Classify 8+ attack vectors with >98% accuracy
  • Integrate with DSRC / C-V2X standards
  • Explainable AI (SHAP) per detection
  • Validate on CARLA + real datasets
Contribution / Novelty
  • Hybrid XGBoost + LSTM ensemble for temporal-spatial detection
  • Built-in SHAP explainability per flagged vehicle
  • Edge runtime via TensorRT — sub-30ms inference
  • Novel kinematic plausibility features beyond PKI
  • Validated in CARLA with attack injection
